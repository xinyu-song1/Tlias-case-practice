Tlias-case-practice

这是一个模拟实现"Tlias智能学习辅助系统"员工信息管理模块的前端练习项目，主要用于巩固HTML的核心知识点

核心功能
1. 员工信息管理：通过表格展示员工姓名、性别、职位、入职日期等信息
2. 条件查询筛选：支持按姓名、性别、职位进行组合查询，可一键清空筛选条件
3. 数据操作入口：提供编辑、删除等操作按钮，模拟完整的管理流程
4. Flex弹性布局：通过 `display: flex`、`justify-content` 实现导航栏与内容区的灵活排布
5. 表单交互：使用 `<input>`、`<select>` 等标签实现查询条件的收集与提交

学习知识点
Flex弹性布局与盒子模型的实际应用
表单与表格的结构化HTML编写
基础的前端交互逻辑实现
响应式页面布局适配
